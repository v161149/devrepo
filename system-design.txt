List of input parameters
 1. The fuel variable (holds the amount of fuel remaining).
 2. The time variable (holds the number of seconds aloft). 
 3. The altitude variable (holds the current altitude in meters).
 4. The velocity variable (holds the current velocity in meters per second).

System architecture

 1. AI core - DeepLearningNetwork (LunarLander)
    build neural network
 2. AI training Deep Learning Network (LunarLander)
    train neural network
 2. AI AISelfPilotProcessor (NeuralPilot)
    apply the trained neural network to deal with real-time rocket decending
 3. AI performanceMetric (PilotScore)
    evaluate the ai performance
 2. AI Rocket Landing Simulator (LanderSimulator) - provide basic system parameters
 ----------------------------
                   AIDeepLearningNetworkCreator
LanderSimulator -> RocketThrottleController
NeuralPilot     -> AISelfPilotProcessor
PilotScore      -> AISelfPilotPerfMetric
LunarLander     -> RocketSelfLandingSimulation
